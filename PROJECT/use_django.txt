1. Python 3.4.
Wenn du ein Mac benutzt, oder eine frühere Version von Python nutzt, möchtest du das Alias "python3" nutzen.

2. Installiere "pip"
Ubuntu/Linux Mint:
sudo apt-get install python3-pip
Mac:
Ab Python 3.4 sollte pip3 mitinstalliert sein.

3. Installiere Django
pip install Django
pip3 install Django

4. Starte den Entwicklungsserver im PROJECT/pizza Verzeichnis:
python manage.py runserver
python3 manage.py runserver
(Erreichbar unter 127.0.0.1:8000)

5. Teste die App im PROJECT/pizza Verzeichnis:
python manage.py test
python3 manage.py test
